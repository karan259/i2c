i2c

Description:

i2c provides an extensive set of utilities (e.g. logging, and options parsing) that unify all RoadNarrows software packages.

Source documentation: i2c-srcdoc

RN Dependencies: None

See also: libserial, librnr

Installation:

Via apt-get (currently available for Ubuntu 12.04 64-bit)

wget -O - http://roadnarrows.com/packages.roadnarrows.key | sudo apt-key add -
sudo sh -c "echo 'deb [arch=amd64] http://roadnarrows.com/distro/apt precise main' > /etc/apt/sources.list.d/rnr.sources.list" 
sudo apt-get update && sudo apt-get install i2c

From source (using CMake)

git clone http://github.com/roadnarrows-robotics/i2c
cd i2c
mkdir build
cd build
cmake ..
make && make install
