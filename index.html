Description:

librnr provides an extensive set of utilities (e.g. logging, and options parsing) that unify all RoadNarrows software packages.

Check out examples in the Source Documentation

Installation:

Via apt-get (currently available for Ubuntu 12.04 64-bit)

wget -O - http://roadnarrows.com/packages.roadnarrows.key | sudo apt-key add -
sudo sh -c "echo 'deb [arch=amd64] http://roadnarrows.com/distro/apt precise main' > /etc/apt/sources.list.d/rnr.sources.list" 
sudo apt-get update && sudo apt-get install librnr-dev

From source (using CMake)

git clone http://github.com/roadnarrows-robotics/librnr
cd librnr
mkdir build
cd build
cmake ..
make && make install
